\subsection{Detailed Description}
Classe permettant le dessin d'un modèle 3\-D sans aucun storage de données ou de commandes sur la carte graphique. 

\begin{DoxyNote}{Note}
Précondition \-: prends pour acquis que le pointeur vers le modèle 3\-D reste valide durant toute la durée de vie du storage.
