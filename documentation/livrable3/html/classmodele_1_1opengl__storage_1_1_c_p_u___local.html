Detailed Description

Classe permettant le dessin d'un modèle 3D sans aucun storage de données ou de commandes sur la carte graphique.

Note
Précondition : prends pour acquis que le pointeur vers le modèle 3D reste valide durant toute la durée de vie du storage.

Martin Paradis

Constructor & Destructor Documentation

modele::opengl_storage::CPU_Local::CPU_Local ( Modele3D const *  modele)

Constructeur avec un modèle 3D.

Construit un storage local pour le dessin d'un modèle 3D. Ne charge aucune donnée sur la carte graphique et utilise les commandes de dessins (dépréciées) d'OpenGL pour dessiner le modèle 3D.

Parameters
[in]modele: le modèle à dessiner.
Returns
Aucune.
